# Fashion-Archetype

## 1. Название проекта
Fashion Archetype — веб-приложение, которое помогает пользователю определить свой стиль одежды через призму психологических архетипов

## 2. Описание проекта
Fashion Archetype — это интерактивный онлайн-тест, предназначенный для людей, которые хотят лучше понять себя и выразить свою личность через одежду. Программа на основе теории 12 архетипов Юнга предлагает персонализированное описание стиля, ключевые элементы гардероба и примеры образов

Основные функции:
*   Прохождение краткого психологического теста
*   Определение архетипа пользователя
*   Предоставление детального описания стиля, соответствующего архетипу
*   Визуальные примеры образов (наброски или описания)

## 3. Установка и запуск

1.  Склонируйте репозиторий:
   
   git clone https://github.com/ulyananenasheva584-collab/fashion-archetype.git
   
   cd fashion-archetype
    

2. npm install

4.  Запустите сервер:
    npm run dev

    
5.  Откройте браузер и перейдите по адресу, который указан в терминале

## 4. Примеры использования
1.  Откройте главную страницу
2.  Нажмите кнопку "Начать тест"
3.  Ответьте на все вопросы, выбрав наиболее подходящий для вас вариант
4.  На странице результата вы увидите ваш архетип, например, "Бунтарь"
5.  Ниже будет представлено описание стиля для Бунтаря
